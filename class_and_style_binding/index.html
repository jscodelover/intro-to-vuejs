Class and Style Binding

Both class and style are attribute of elements. They use v-bind vuejs directive.

Class Binding

#Object Syntax

1. The above syntax means the presence of the "activeClass" class will be determined by the truthiness of the data property "isActive".

<div v-bind:class="{activeClass: isActive}"></div>

const app = new Vue({
  el: "#root",
  data:{
   isActive: false,
   activeClass: 'heading'
  }
})

2.The bound object doesn’t have to be inline: 2. You can have multiple classes toggled by having more fields in the object. In addition, the v-bind:class directive can also co-exist with the plain class attribute.

<div class="static" :class="{ activeClass: isActive, 'text-danger': hasError }"></div>

3.The bound object doesn’t have to be inline:

<div :class="classObject"></div>

const app = new Vue({
  el: "#root",
  data:{
   classObject: {
     'active': false,
     'text-danger': true,
   }
  }
});

#Array Syntax

1. We can pass an array to v-bind:class to apply a list of classes.

<div v-bind:class="[activeClass, errorClass]"></div>

const app = new Vue({
  el: "#root",
  data:{
   activeClass: 'active',
   errorClass: 'text-danger'
  }
});

2. You can do it with a ternary expression:

<div v-bind:class="[isActive ? activeClass : '', errorClass]"></div>

3. However, this can be a bit verbose if you have multiple conditional classes. That’s why it’s also possible to use the object syntax inside array syntax:

<div v-bind:class="[{ activeClass: isActive }, errorClass]"></div>

Style Binding

#Object Syntax

It's same as class binding.

<div v-bind:style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>

const app = new Vue({
  el: "#root",
  data:{
   activeColor: 'red',
   fontSize: 30
  }
})

Style Object.

<div v-bind:style="styleObject"></div>

const app = new Vue({
  el: "#root",
  data:{
   styleObject: {
     activeColor: 'red',
     fontSize: 30,
   }
  }
});

#Array Syntax

<div v-bind:class="[baseStyles, overridingStyles]"></div>
